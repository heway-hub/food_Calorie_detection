# NutriScan - 食物卡路里检测与记录系统

基于 Python 的桌面应用程序，用于食物图像识别、卡路里计算和饮食记录管理。

## 功能特点

- 📸 食物图片上传与识别
- 🔍 自动识别食物类型和份量
- 📊 卡路里和营养成分计算
- 📝 饮食记录管理
- 📈 数据可视化和趋势分析
